Q-1:What is the difference between map, filter,forEach,find?

1.map() :The map() method creates a new array with the results of calling a provided function on every element in the original array. The function is applied to each element in the array, and the resulting value is added to the new array. The map() method does not modify the original array.

2.find() :The find() method returns the value of the first element in the array that satisfies the provided testing function. If no elements in the array satisfy the testing function, undefined is returned

3.filter() : The filter() method creates a new array with all elements that pass the test implemented by the provided function. The function is applied to each element in the array, and if it returns true, the element is included in the new array.

4.forEach() : The forEach() method executes a provided function once for each array element. The function is applied to each element in the array, but it does not return a new array.

Q-2:What is the difference between Local Storage, Session Storage, Coociks?

1.local Storage :local Storage is a web browser API that provides a way to store data on the client-side, meaning within the user's web browser. It is a type of persistent storage, which means the data remains even after the user closes the browser window or restarts the computer.

2.Session storage :Session storage is a web browser API that provides a way to store data on the client-side, meaning within the user's web browser, but unlike local storage, the data is only available for the duration of the user's browsing session. This means that the data is lost once the user closes the browser window or tab.

3.Cookies : cookies are small text files that are stored on a user's computer by a website or web application. Cookies are created by a web server when a user first visits a website, and they are stored on the user's browser for subsequent visits.
